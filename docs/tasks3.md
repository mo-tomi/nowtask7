## チェックリストの使い方

各タスクの左側の [ ] にチェックマーク [x] を入れて、完了状況を記録してください。

例：[x] ページのタイトルを設定 ← 完了
例：[ ] ページのタイトルを設定 ← 未完了

すべてのタスクが完了すると、その上の優先度関連のセクションが完成します。
